핵심 기능

JLPT N2 시험 준비를 위해 필요한 모든 걸 한 앱에 담았어요.

📚

회차별 풀이

실제 N2 모의고사 11회차. 어휘·문법·독해·청해 100문제씩 회차 단위로 학습.

🎯

영역별 모아풀기

한자읽기, 문맥 규정, 장문 독해 등 18개 영역별로 11회차 분량을 한 자리에서.

🎧

청해 mp3 내장

55개 청해 음원 모두 앱에 번들. 인터넷 없이 어디서나 듣고 풀 수 있어요.

⭐

단어장 + 외우기

모르는 단어 ★ 표시 → 플래시카드로 알아요/몰라요 반복. 다 알 때까지 루프.

🔤

한자 음·뜻

단어 카드와 사전 시트에서 구성 한자의 음(파랑)과 뜻(회색)을 함께 표시.

📝

한국어 해설

모든 문제에 한국어 해설. 정답·핵심 이유·오답 분석으로 블록 단위 표시.

🔒

완전 오프라인

네트워크 권한 없음. 진도·단어장 모두 기기 안에만 저장 — 데이터 전송 0.

📊

진도·정답률 추적

회차/영역별 풀이율과 정답률을 한눈에. 이어서 풀기로 끊김 없는 학습.
