개인정보처리방침

시행일: 2026년 5월 20일 · 최종 업데이트: 2026년 5월 20일
요약 — 엔투노트는 어떤 개인정보도 수집하지 않고, 외부로 전송하지 않으며, 인터넷 통신을 사용하지 않습니다. 모든 학습 데이터는 사용자 기기 안에만 저장됩니다.

1. 수집하는 정보

엔투노트는 이름·이메일·전화번호·생년월일 등 어떤 식별 정보도 수집하지 않습니다. 회원 가입 절차가 없으며, 외부 서버와 통신하지 않습니다.

기기 내부에만 저장되는 데이터

  • 학습 진도 — 회차별 풀이 결과 (정답/오답, 선택지, 시각)
  • 청해 진도 — mondai별 답안과 결과
  • 단어장 — 사용자가 ★ 표시한 단어 목록
  • 학습 통계 (SRS) — 단어별 알아요/몰라요 횟수와 마지막 학습 시각
  • 앱 설정 — 후리가나 토글, 탭 위치 등 UI 상태

이 데이터는 iOS/Android 의 SharedPreferences(=기기 로컬 키-값 저장소) 에만 기록되며, 앱을 삭제하면 모두 영구적으로 제거됩니다.

2. 정보 전송

엔투노트는 인터넷 권한을 요청하지 않으며(Android INTERNET 퍼미션 없음, iOS ATS 예외 없음), 외부 서버·분석 도구·광고 SDK 와 통신하지 않습니다.

  • 광고 없음 (광고 ID 사용 안 함)
  • 분석 SDK 없음 (Firebase / Google Analytics 등 미사용)
  • 크래시 리포팅 없음
  • 위치 정보·연락처·사진·마이크 등 어떤 권한도 요청하지 않음

3. 학습 콘텐츠

앱에 포함된 일본어 문제·청해 음원·해설은 모두 앱 번들에 포함되어 있으며 사용자 디바이스 안에서 재생됩니다. 외부에서 추가로 다운로드하는 콘텐츠는 없습니다.

4. 어린이의 개인정보

만 14세 미만 어린이의 개인정보를 별도로 수집하지 않습니다. 본 앱은 모든 사용자에게 동일하게 작동하며, 어떤 사용자로부터도 개인정보를 수집하지 않습니다.

5. 정보 보유 기간

앱 내 데이터는 사용자가 앱을 삭제하거나 앱 내 "초기화" 기능을 사용할 때까지 기기에만 보관됩니다. 외부 백업이나 동기화는 제공하지 않습니다.

6. 권리

엔투노트는 사용자의 개인정보를 보유하지 않으므로 별도의 열람·정정·삭제·전송 요청 절차가 필요하지 않습니다. 모든 학습 데이터는 기기 내 앱 설정에서 직접 확인하고 "진도 초기화" 또는 앱 삭제로 직접 제거할 수 있습니다.

7. 변경

본 방침이 변경되는 경우 본 페이지 상단의 "최종 업데이트" 날짜를 갱신합니다. 중요한 변경 사항이 있을 경우 앱 출시 노트나 본 페이지에 안내합니다.
